# Codex Assistant User Guide

## Quick Start

### Opening the Tool Window
- **Keyboard**: Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux)
- **Menu**: View → Tool Windows → Codex Chat

### Basic Usage
1. Type your question in the input area
2. Press "Run" or hit Enter to send
3. View the AI response in the timeline

## Smart Context

### Auto-Add Open Files
Enable "Auto-add open files" checkbox to automatically include files you're currently editing.

### Git Diff Integration
Enable "Include git diff" to add uncommitted changes to the context.

### Manual File Attachment
Click "Attach File" to manually select additional files.

## Code Changes

### Viewing Diffs
When AI suggests code changes, use the diff viewer to see before/after comparison.

### Applying Changes
Click "Apply" to write changes directly to your files.

## Keyboard Shortcuts

- `Cmd+K` / `Ctrl+K`: Open tool window
- `Cmd+Enter` / `Ctrl+Enter`: Send message
- `Esc`: Cancel running request

## Tips

- Keep context files relevant to reduce token usage
- Use git diff for debugging recent changes
- Review diffs carefully before applying
- Start new sessions for unrelated tasks
